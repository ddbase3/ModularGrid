ModularGrid - Infinite Ajax Workbench

This demo keeps the ajax source and composes the grid mainly through plugins and configuration: search, filters, header menus, info, selection, row actions, bulk actions, export, column visibility, reset, row detail, storage and automatic append loading through the infinite-scroll plugin.

The table view uses a fixed-height internal scroll area. It loads the first 50 records and automatically requests the next batch when the scroll position approaches the bottom. Clicking a row opens a server-loaded detail panel with extra content that is not rendered in the main list.
