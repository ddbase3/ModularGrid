ModularGrid - Grouping Infinite Ajax

This demo starts from the same workbench-style baseline as the infinite ajax demo: search, filters, header menus, selection, row actions, bulk actions, export, column visibility, reset, storage, async row detail and automatic append loading.

On top of that, you can switch grouping fields on and off live. With no grouping selected, the grid behaves like the normal infinite table. Once grouping fields are active, the server returns grouped rows and the expanded detail layer renders the matching child entries as a child table.
